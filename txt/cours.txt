Initialiser un dépôt , Git git init
-- Suivre les fichiers 
Ajouter, git add fichier.txt
Ajouter tout les fichier ,git add .
voir les status,git status
Enregistrer les changements (commit), git commit -m "Message clair qui explique la modification"
Envoyer vers GitHub (push),git push
Envoyer premier foix, git push --set-upstream origin main 
Récupérer les changements (pull) ,git pull

-- les branches 
Gérer les branches,git branch
cree un nouveau branch ,   git branch feature/test
aller au branch,git checkout feature/test
aller et cree au meme temps ,git checkout -b feature/test
-- Fusionner les changement 
Fusionner les changement ,git checkout main
integre un branch, git merge feature/test
-- Voir l’historique
Voir l’historique ,git log


-- Résumé des commandes
-- git init               # Initialiser projet Git
-- git status             # Voir état fichiers
-- git add .              # Ajouter tous les fichiers
-- git commit -m "msg"    # Sauvegarder changements
-- git push               # Envoyer vers GitHub
-- git pull               # Récupérer dernières modifs
-- git branch             # Voir branches
-- git checkout -b nom    # Créer + switch nouvelle branche
-- git merge nom          # Fusionner branche dans main
